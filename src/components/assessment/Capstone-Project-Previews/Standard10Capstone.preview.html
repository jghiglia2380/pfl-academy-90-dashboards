Housing Decision Analysis Project

Standard 10: Housing Decisions

45 Minutes Calculations Analysis

In this capstone project, you will analyze different housing scenarios by first performing detailed calculations, then using those results to make and justify housing recommendations based on your stated priorities and assumptions.

Required Materials:

  • Calculator
  • Worksheet or digital document
  • Spreadsheet software (optional)

Project Tasks

Housing Market Analysis (15 minutes)

Research and analyze housing options in both scenarios

→ Review current market conditions

→ Compare location advantages

→ Analyze property value trends

→ Evaluate neighborhood factors

→ Document market findings

Financial Analysis (15 minutes)

Calculate and compare all housing costs

→ Calculate monthly payments

→ Compare total upfront costs

→ Project long-term expenses

→ Analyze investment potential

→ Assess affordability ratios

Decision Analysis (15 minutes)

Develop final recommendations with justification

→ Evaluate lifestyle fit

→ Assess future flexibility

→ Consider market timing

→ Create action timeline

→ Present recommendations
